# Security Policy

## Reporting

Report vulnerabilities privately to the `abuzucom` maintainers. Do not open a
public issue containing credentials, bypass details, or an exploitable proof of
concept.

## Scope

This repository protects the integrity of pull-request review for consuming
repositories. It does not replace branch protection, CODEOWNERS, dependency
review, secret scanning, or least-privilege GitHub token permissions.

## Maintainer rule

Changes to the checker, reusable workflow, local action, policy format, or
installation templates are security-sensitive and require review by
`itsjustatank`.
